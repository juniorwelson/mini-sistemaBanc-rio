# 💳 SystemBank - Sistema Bancário em Kotlin

Projeto desenvolvido em **Kotlin** com o objetivo de praticar conceitos fundamentais da linguagem e da **Programação Orientada a Objetos (POO)**.

O SystemBank simula um sistema bancário com operações básicas, permitindo criação de contas, depósitos, saques e gerenciamento de contas, aplicando boas práticas de organização e estruturação de código.

---

## 🚀 Funcionalidades

✅ Criação de conta corrente  
✅ Criação de conta poupança  
✅ Depósitos  
✅ Saques com validações  
✅ Consulta de saldo  
✅ Criação e validação de senha  
✅ Validação de dados do usuário  
✅ Menu interativo via terminal  

---

## 🧠 Conceitos Aplicados

Durante o desenvolvimento foram aplicados conceitos importantes do Kotlin:

- 🧩 Programação Orientada a Objetos (POO)
- 🔒 Encapsulamento
- 🧬 Herança
- 🎯 Abstração
- 🔄 Separação de responsabilidades
- ⚠️ Null Safety do Kotlin
- 🧹 Clean Code
- 📂 Organização de pacotes e classes
